Algunos lenguajes de la web

HTML + CSS + Javascript

  • HTML 5

    • HTML5 (HyperText Markup Language, versión 5) es la quinta revsisión importante del lenguaje básico de la World Wide Web, HTML.

    Usa la extensión .html

  • CSS 3

    • Hoja de estilo en cascada o CSS (siglas en inglés de cascading style sheets) es un lenguaje para definir y crear la presentación de un documento estructurado escrito en HTML.

    Usa la extensión css

  • Javascript

    • Javascript (abreviado comúnmente "JS")es un lenguaje de programación interpretado utilizado principalmente en el lado del cliente (client-side), implementado como parte de un navegador web permitiendo mejoras en la interfaz del usuario y páginas web dinámicas.

    Usa la extensión .js
